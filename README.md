
Run the ```run.sh``` file:
```bash
./run.sh
```
Output: 
* datasets in  ```data/``` folder and ```compiled_data/``` folder

#### To Generate File Information onto feature_table.csv
Run the ```[something].sh``` file:

```bash
./[something].sh
```

```bash
python generate_scripts.py
```
