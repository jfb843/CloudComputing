#### To Download Datasets
Run the ```datasets.sh``` file:
```bash
./datasets.sh
```

* [Kaggle-api](https://github.com/Kaggle/kaggle-api)

## Generating Labels
The labels (runtimes) for each file can be generated by running the ```generate_labels.sh``` file:
```bash
./generate_labels.sh
```
